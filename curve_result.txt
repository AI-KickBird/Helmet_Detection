<결과 분석>

`yolov8n 모델학습`결과

1. Confusion Matrix - 모델이 예측한 클래스와 실제 클래스를 비교하여 성능을 평가
    - 행 : 실제 클래스
    - 열 : 모델의 예측 클래스
    - 잘못된 예측은 대각선 이외의 위치에 나타난다.

2. Normalized Confusion Matrix - 각 값이 전체 데이터의 비율로 나타나 상대적 중요도를 알 수 있다.

3. F1_curve
- **X축 (Confidence)**:
    - 모델이 탐지한 바운딩 박스의 신뢰도(Confidence) 값.
    - 예: Confidence = 0.4 → 모델이 특정 객체가 해당 클래스일 확률이 40%라고 판단.
- **Y축 (F1 점수):**
    - 모델의 F1 점수로, Precision(정밀도)과 Recall(재현율)의 조화 평균.
    - F1 점수가 높을수록 탐지가 더 정확하고 민감하게 이루어짐.
